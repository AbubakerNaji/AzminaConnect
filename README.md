# Prime Era - Futuristic IT Company Landing Page

A modern, bilingual (English/Arabic) landing page for Prime Era, featuring a futuristic design with cyber-cyan, silver, and black color palette.

## Features

- **Futuristic Logo**: Animated hourglass morphing into digital pixels, representing time and speed
- **Bilingual Support**: Full English and Arabic language support with RTL layout
- **Responsive Design**: Mobile-first approach with smooth animations
- **Modern Tech Stack**: Pure HTML5, CSS3, and Vanilla JavaScript
- **Smooth Scrolling**: Enhanced navigation experience
- **Interactive Elements**: Animated service cards, form interactions, and scroll effects

## Design Elements

### Logo Concept
The Prime Era logo combines two concepts:
1. **Hourglass**: Represents time and speed
2. **Pixel Grid Overlay**: Digital transformation
3. **Fiber Optic Number "1"**: Alternative visual in hero section showing the number "1" made of fiber optic lines

### Color Palette
- **Cyber Cyan** (#00ffff): Primary accent color
- **Silver** (#c0c0c0): Secondary text color
- **Black** (#000000): Background color

## Sections

1. **Header**: Fixed navigation with logo and language toggle
2. **Hero**: Main landing section with animated visuals
3. **Services**: Grid of service cards with icons
4. **About**: Company information with statistics
5. **Contact**: Contact form and information
6. **Footer**: Links and company information

## Usage

1. Open `index.html` in a modern web browser
2. Use the language toggle button (EN/AR) to switch between English and Arabic
3. Navigate through sections using the menu or scroll
4. The page is fully responsive and works on all devices

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

### Changing Colors
Edit the CSS variables in `styles.css`:

```css
:root {
    --cyber-cyan: #00ffff;
    --silver: #c0c0c0;
    --black: #000000;
    /* ... */
}
```

### Adding Content
All text content uses `data-en` and `data-ar` attributes for bilingual support. Add translations in the `translations` object in `script.js`.
